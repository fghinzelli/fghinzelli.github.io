### Cache commands

```
php vendor/bin/doctrine orm:generate-proxies

// Clear cache
php vendor/bin/doctrine orm:clear-cache:metadata
php vendor/bin/doctrine orm:clear-cache:query
php vendor/bin/doctrine orm:clear-cache:region:collection
php vendor/bin/doctrine orm:clear-cache:region:entity
php vendor/bin/doctrine orm:clear-cache:region:query
php vendor/bin/doctrine orm:clear-cache:result
```
